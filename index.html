OHSU Comp. Bio.

Funnel runs your tasks.

Given a task, Funnel will find a worker in a cluster, download input files, run the task in Docker, upload output files, and capture logs.

Download v0.1 Learn more

Funnel is alpha quality.

Funnel is a young project and there's still a lot to do. It's usable, but important things are broken and missing.

Check it out.

Start Funnel

$ funnel server

Run a task

Returns a task ID.

$ funnel run 'md5sum $src' -I ubuntu --in src=~/src.txt
b41pkv2rl6qjf441avd0

Get the task

Returns state, logs, and more.

$ funnel task get b41pkv2rl6qjf441avd0

List all the tasks

$ funnel task list

View the terminal dashboard

$ funnel dashboard

Use a remove server

$ funnel run --server http://funnel.example.com ...

Example tasks

$ funnel example list
$ funnel example hello-world

Get help

$ funnel help

Get the code

$ go get github.com/ohsu-comp-bio/funnel
